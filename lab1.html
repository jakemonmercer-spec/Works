Программирование расчета простых и сложных процентов

Дисциплина: Программирование кредитных отношений и банковское дело

1. Теоретическая база

В рамках работы реализованы два базовых метода начисления процентов, используемых в банковских продуктах:

  • Простые проценты: Начисляются только на основную сумму вклада.
  • Сложные проценты: Начисляются на основную сумму плюс накопленные за предыдущие периоды проценты (капитализация).

2. Реализация на Python

В отличие от стандартного использования Excel, реализация на Python позволяет автоматизировать массовые расчеты и легко интегрировать их в банковские системы.

def simple_interest(p, r, t):
    # P - сумма, R - ставка (%), T - срок (лет)
    return p * (1 + (r / 100) * t)

def compound_interest(p, r, t, n=12):
    # n - количество периодов капитализации в год (по умолчанию 12)
    return p * (1 + (r / 100) / n)**(n * t)

# Пример расчета
principal = 100000  # 100 000 руб.
rate = 15           # 15% годовых
years = 5           # на 5 лет

print(f"Простые: {simple_interest(principal, rate, years)}")
print(f"Сложные: {compound_interest(principal, rate, years)}")

4. Вывод

Использование Python для расчетов в банковской сфере дает преимущество в точности и гибкости. Сложные проценты значительно выгоднее для долгосрочных вкладов, что наглядно демонстрирует расчет: за 5 лет разница в доходности составила более 35% от первоначальной суммы.
