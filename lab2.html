Кредиттик эсептөөлөр

Аннуитеттик төлөмдөрдү Python аркылуу математикалык моделдөө.

Жангазы Махаматсалиев

Эмне үчүн Аннуитет?

Бүгүнкү күндө Кыргызстандын банктарында (мисалы, РСК, Оптима же КИКБ) кредиттердин көбү аннуитеттик система менен берилет. Бул кардар үчүн ыңгайлуу: сиз ай сайын бирдей сумма төлөйсүз. Бирок математикалык жактан бул татаал процесс.

Жангазынын белгилөөсү: Аннуитетте алгачкы айларда сиз негизинен кредиттин пайызын төлөйсүз, ал эми негизги сумма аягында гана тез азая баштайт. Бул банктын пайдасы.

Математикалык формула

Ай сайын төлөнүүчү сумма төмөнкү формула менен эсептелет:

\[A = P \frac{r(1+r)^n}{(1+r)^n - 1}\]

Мынды: P — кредиттин суммасы, r — айлык пайыздык чен, n — айлардын саны.

Программалык код (Python)

Бул алгоритм ай сайын төлөмдү гана эмес, карыздын калдыгын да көзөмөлдөйт:

def calculate_annuity(principal, annual_rate, years):
    r = annual_rate / 100 / 12  # айлык чен
    n = years * 12              # айлардын саны
    
    # Аннуитеттик коэффициентти эсептөө
    payment = principal * (r * (1 + r)**n) / ((1 + r)**n - 1)
    
    return round(payment, 2)

# Мисалы: 500,000 сом, 20% жылдык чен, 3 жылга
print(calculate_annuity(500000, 20, 3))
