Расчет аннуитетного кредита

Разработка алгоритма ежемесячных платежей и графиков погашения.

Студент: Махаматсалиев Жангазы

Что такое аннуитет?

Аннуитетный платеж — это вариант выплат по кредиту, когда ежемесячный взнос остается неизменным на протяжении всего срока. Это самая популярная схема в банках Кыргызстана. Однако внутри платежа пропорция меняется: в начале вы платите больше процентов, а в конце — основной долг.

Математическая модель

Расчет коэффициента аннуитета производится по формуле:

\[A = P \frac{r(1+r)^n}{(1+r)^n - 1}\]

Где: P — сумма кредита, r — процентная ставка в месяц, n — срок в месяцах.

Реализация на Python

Программный расчет позволяет автоматизировать вывод графиков для тысяч клиентов одновременно.

def calculate_annuity(p, annual_rate, years):
    # Месячная ставка
    r = annual_rate / 100 / 12
    # Срок в месяцах
    n = years * 12
    
    # Формула аннуитета
    payment = p * (r * (1 + r)**n) / ((1 + r)**n - 1)
    
    return round(payment, 2)

# Пример: 500,000 сом, 22% годовых на 3 года
print(f"Платеж: {calculate_annuity(500000, 22, 3)} KGS")

Вывод
